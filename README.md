# Lucky Wheel

一个可自定义选项的幸运转盘应用。

## 功能特点

- 可自定义转盘选项
- 支持添加、删除、修改选项
- 选项数量不固定
- 平滑的旋转动画效果
- 响应式设计，支持移动端

## 本地运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 在浏览器中打开 http://localhost:3000

## 部署到 Vercel

1. 将代码推送到 GitHub 仓库
2. 在 Vercel 中导入项目
3. Vercel 会自动检测并部署静态网站

## 使用方法

1. 在"编辑选项"区域添加或修改转盘选项
2. 点击"开始旋转"按钮
3. 等待转盘停止，查看结果
